#  Eventify – Event & Resource Management Platform

**Eventify** is a modern web-based platform designed to streamline **event coordination** and **resource booking** for college clubs and campus organizations. Built on the **MERN stack** and enhanced with **Generative AI**, Eventify offers a centralized, role-based system to manage events, book venues, track participation, and gather feedback—all in one place.

---

##  About
In campus environments where multiple clubs organize events year-round, overlapping schedules, resource conflicts, and manual approvals can cause inefficiencies. **Eventify** solves these problems with a **centralized, AI-powered platform** that:
- Simplifies **event proposal, approval, and scheduling**
- Automates **resource allocation** in real-time
- Provides **personalized event recommendations**
- Offers an **analytics dashboard** for better decision-making

---

##  Features
- **Role-Based Access Control (RBAC)** for Admins, Organizers, and Students
- **Dynamic Event Approval Workflow** with status tracking
- **AI-Powered Event Recommendations** based on trends and user preferences
- **AI Chatbot Assistant** for event discovery and FAQs
- **Real-Time Resource Booking** with conflict prevention
- **Interactive Calendar View** (weekly & monthly)
- **Feedback & Analytics Module** with participation trends
- **Responsive UI** designed with Tailwind CSS

---

##  Tech Stack

**Frontend**
- React.js
- Tailwind CSS

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (MongoDB Atlas for cloud hosting)

**AI Integration**
- OpenAI API (for event ideas, recommendations, and chatbot)

---

##  System Architecture
- **Frontend (React)** communicates with **RESTful APIs** in Express.js
- **MongoDB** stores users, events, resources, and feedback
- **JWT Authentication** ensures secure role-based access
- **Real-Time Updates** for events and resource booking
- **AI Modules** integrated into backend for recommendations and chatbot

---

##  AI Integration
- **Event Suggestions**: Generates creative event names, themes, and descriptions
- **Personalized Recommendations**: Suggests events tailored to user interests
- **AI Chatbot**: 24/7 interactive assistant for event discovery & registration help
